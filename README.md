# PantherX Desktop WIKI Application

A small/embedded browser for loading the `px-wiki` html files.


## Scheme Url
You can open your related pages in `px-wiki` via scheme-url:

```
$ xdg-open px-desktop-wiki:page=npm/index.html
```

```
$ xdg-open px-desktop-wiki:page=qemu/index.html#boot-from-iso
```
